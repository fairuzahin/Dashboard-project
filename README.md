# Excel Store Sales Dashboard (2023)

This project is an interactive Excel dashboard created to analyze store sales performance for the year 2023.  
The dashboard provides insights into sales trends, customer demographics, order status, top-performing states, and sales channels.

This project was created as a guided learning exercise using a YouTube tutorial to practice data analysis and dashboard design skills.

---

## Tools Used
- Microsoft Excel  
- Pivot Tables  
- Pivot Charts  
- Slicers (Filters)  
- Data Cleaning  
- Data Visualization  

---

## Dashboard Overview

The dashboard includes the following sections:

### 1. Order vs Sales Trend
Shows monthly sales revenue and order volume to identify performance trends over time.

### 2. Sales by Gender
Compares total sales between male and female customers.

### 3. Order Status Distribution
Displays the percentage of Delivered, Cancelled, Returned, and Refunded orders.

### 4. Top 5 States by Sales
Highlights the highest revenue-generating states.

### 5. Age vs Gender Analysis
Breaks down order percentages by age group (Adult, Senior, Teenager) and gender.

### 6. Sales Channels
Shows order distribution across platforms such as Amazon, Flipkart, Ebay, Shein, Temu, and others.

---

## Key Insights

- Most orders were successfully delivered (over 90%).
- Female customers generated higher sales than male customers.
- Georgia and Texas were the top-performing states by revenue.
- Amazon and Ebay were the leading sales channels.
- Adults placed the highest number of orders across both genders.
